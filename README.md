# Sunshine Corp

Boilerplate code for Sunshine Corp. (fullstack interview)

## Installation

```
npm run setup
```

OR

```
cd backend && npm i
cd client && npm i
```

## Run both BE and FE

To run both the back end and the front end run

```
npm run dev
```

## Individually

```
cd backend && npm run dev
cd client && npm run dev
```
